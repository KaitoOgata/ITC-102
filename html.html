About HTML

HTML stands for hyper text markup language. And it is used to express webpages at

world wide web. HTML has function of link, images, document, and design. HTML can embed

programs written in a Scripting language such as JavaScript, which affects the behavior

content of web pages. Web browser receive HTML documents from a web server or from local

storage and render the documents into multimedia web pages. HTML describes the structure of a

web page semantically and originally included cues for the appearance of the document.
